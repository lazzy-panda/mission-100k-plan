## Рабочие каналы Russia → Georgia (проверено апр. 2026)

### 🟢 Главный канал — Tinkoff IBAN → Грузия

**Источник: официальная справка Т-Банк** (https://www.tbank.ru/bank/help/cross-border/directions/v/ge/)

| Параметр | Значение |
|----------|----------|
| Банки в Грузии | Bank of Georgia, TBC, **Credo, Liberty**, Basisbank |
| Валюта отправки → получения | RUB → GEL (конвертация в пути) |
| Лимит на операцию | 3,500 — **600,000 ₽** |
| Комиссия (базовый) | 0% до 20K, далее 1% от превышения |
| Комиссия (Premium) | 0% до 100K, далее 1% |
| Срок | 1-2 рабочих дня |
| Доступ | Только мобильное приложение Тинькофф |

**Под твой объём (800K ₽/мес):** 2 операции по 400K в месяц. С Premium (~990 ₽/мес подписка) — 8 операций по 100K без комиссии. Premium окупается с лихвой.

**Минус:** двойная конверсия RUB → GEL → EUR теряет 2-4% на спредах. Решение — попросить Credo сразу держать multi-currency счёт (GEL + USD + EUR), конвертить GEL→EUR внутри банка по их курсу.

### 🟡 Backup #1 — USDT-мост
Если IBAN-канал даст сбой или нужны суммы > 600K разово:
```
Tinkoff RUB → P2P-биржа (Bybit/Garantex) → USDT TRC-20
            → грузинская биржа (CryptoExchange.ge) → USD/EUR
            → Credo bank account → Revolut SWIFT
```
**Плюсы:** без лимитов на операцию, одна конверсия. **Минусы:** P2P-риски, сложнее.

### 🟡 Backup #2 — Золотая Корона
Tinkoff → Корона (онлайн) → cash в Грузии по загранпаспорту → класть на счёт. **$5-10K за операцию, комиссия 1-1.5%.** Только наличными, потом в банк руками.

### 🔵 Cash carry (только initial deposit)
$1-3K наличными в первый перелёт для открытия счёта. До $10K без декларации.

### Финальное звено: Georgia → Revolut
Стандартный EU SWIFT из Credo/Liberty в Revolut (IBAN в Литве/UK) — проблем не должно быть.

## Что обсудить с банком на месте

1. **SWIFT прием из РФ** — текущий статус correspondent banking. Каких банков-отправителей принимают (Tinkoff, Сбер, ВТБ, etc.).
2. **Переводы в EUR на Кипр** — комиссии, лимиты, время.
3. **Multi-currency operation** — конверсия RUB → GEL → EUR / USD: курсы, спред, ограничения.
4. **Online/mobile banking** — что доступно, есть ли русскоязычный интерфейс.
5. **Ограничения по операциям** для нерезидентов (есть ли monthly cap, нужна ли отдельная авторизация для крупных переводов).

## Backup план: USDT-мост (если банк-схема не сработает)

Независимо от грузинского счёта, USDT-мост остаётся самым стабильным способом:

- Tinkoff (RUB) → Bybit / Garantex / OKX (P2P покупка USDT TRC-20) → грузинская биржа (CryptoExchange.ge / etc.) → конверсия в USD/EUR → перевод на грузинский счёт ИЛИ напрямую на Кипр

Грузинский банковский счёт + USDT-мост = два независимых канала, что снижает риск sole-point-of-failure.
